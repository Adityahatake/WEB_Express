    DOM (Document Object Model) Navigation in HTML

    The Document Object Model (DOM) is a programming interface for HTML and XML documents. 
    It represents the page so that programs can change the document structure, style, and content. 
    The DOM represents the document as a tree of nodes, where each node is an object representing a part of the document.

    Key Concepts:
    1. Document Root:
        - The root of the DOM tree is the 'document' object.
        - Example: document.body refers to the <body> element.

    2. Node Relationships:
        - Parent Node: The node directly above another node.
        - Child Node: The node directly below another node.
        - Sibling Nodes: Nodes that share the same parent.

    3. Navigating the DOM:
        - Accessing Elements:
            - document.getElementById('id'): Selects an element by its ID.
            - document.getElementsByClassName('class'): Selects elements by class name.
            - document.getElementsByTagName('tag'): Selects elements by tag name.
            - document.querySelector('selector'): Selects the first element matching a CSS selector.
            - document.querySelectorAll('selector'): Selects all elements matching a CSS selector.

        - Traversing Nodes:
            - parentNode: Returns the parent node of the specified node.
            - childNodes: Returns a collection of a node's child nodes (including text and comment nodes).
            - children: Returns a collection of a node's child elements (element nodes only).
            - firstChild / lastChild: Returns the first/last child node.
            - firstElementChild / lastElementChild: Returns the first/last child element.
            - nextSibling / previousSibling: Returns the next/previous sibling node.
            - nextElementSibling / previousElementSibling: Returns the next/previous sibling element.

    4. Example Navigation:
        - Access the first <ul> element:
            var ul = document.querySelector('ul');
        - Access the first <li> in the <ul>:
            var firstLi = ul.firstElementChild;
        - Access the parent of the <li>:
            var parent = firstLi.parentNode;
        - Access the next sibling of the <li>:
            var nextLi = firstLi.nextElementSibling;

    5. Modifying the DOM:
        - You can add, remove, or modify elements and their content using DOM methods.
        - Example: 
            var newLi = document.createElement('li');
            newLi.textContent = 'New Item';
            ul.appendChild(newLi);

    6. Best Practices:
        - Use querySelector/querySelectorAll for flexible CSS-like selection.
        - Prefer children/firstElementChild over childNodes/firstChild to avoid text nodes.
        - Always check for null when navigating (e.g., nextElementSibling may be null).


    Summary:
    DOM navigation allows you to programmatically access and manipulate HTML elements and their relationships. 
    Understanding parent, child, and sibling relationships, as well as the various DOM selection and traversal methods, 
    is essential for effective web development.
