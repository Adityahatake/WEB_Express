JavaScript Operators Explained

1. Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (Remainder)
  • ++ Increment
  • -- Decrement
let a = 10, b = 3;
console.log(a + b); // 13
console.log(a % b); // 1

2. Assignment Operators

  • = Assign
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
let x = 5;
x += 2; // x = 7

3. Comparison Operators

  • == Equal to
  • === Equal value and type
  • != Not equal
  • !== Not equal value or type
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to
console.log(5 == '5');  // true
console.log(5 === '5'); // false

4. Logical Operators

  • && Logical AND
  • || Logical OR
  • ! Logical NOT
let a = true, b = false;
console.log(a && b); // false
console.log(a || b); // true
console.log(!a);     // false

5. Other Operators

  • typeof Type of variable
  • ? Ternary operator
console.log(typeof 123); // "number"
let age = 18;
let canVote = (age >= 18) ? "Yes" : "No";
