JavaScript Math Object Tutorial

The Math object in JavaScript provides mathematical constants and functions. It is a built-in object and does not need to be instantiated.

Common Properties

  • Math.PI — Value of Ï€ (3.14159...)
  • Math.E — Euler's number (2.718...)
  • Math.LN2 — Natural logarithm of 2
  • Math.SQRT2 — Square root of 2
console.log(Math.PI);   // 3.141592653589793
console.log(Math.E);    // 2.718281828459045

Common Methods

  • Math.abs(x) — Absolute value
  • Math.round(x) — Rounds to nearest integer
  • Math.floor(x) — Rounds down
  • Math.ceil(x) — Rounds up
  • Math.max(a, b, ...) — Largest value
  • Math.min(a, b, ...) — Smallest value
  • Math.random() — Random number between 0 and 1
  • Math.pow(x, y) — x to the power of y
  • Math.sqrt(x) — Square root
