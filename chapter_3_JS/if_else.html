If / Else — quick tutorial with examples

These examples show common patterns: if, if...else, if...else if..., ternary, nested if, and truthy/falsy checks. Use the controls and click the buttons to run each example.

1. Basic if

// JavaScript used:
if (n > 0) {
    // runs when n is greater than 0
    result = 'positive';
}

2. if ... else

// JavaScript used:
if (age >= 18) {
    result = 'You can vote';
} else {
    result = 'You cannot vote yet';
}

3. if ... else if ... else (grading)

// JavaScript used:
if (s >= 90) { grade = 'A'; }
else if (s >= 80) { grade = 'B'; }
else if (s >= 70) { grade = 'C'; }
else if (s >= 60) { grade = 'D'; }
else { grade = 'F'; }

4. Ternary operator (short if/else)

// JavaScript used:
result = (n % 2 === 0) ? 'even' : 'odd';

5. Nested if (multiple checks)

// JavaScript used:
if (t > 30) {
    if (t > 40) advice = 'Extreme heat — stay inside';
    else advice = 'Hot — drink water';
} else { advice = 'Pleasant or cool'; }

6. Truthy / falsy example (default values)

// JavaScript used:
const displayName = name || 'Guest'; // if name is falsy (empty string, null, undefined, 0) it falls back
